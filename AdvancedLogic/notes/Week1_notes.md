## 1.1 What are (Non-classical) Logics?

### Formal Languages and Well-formed Formulas

To define a logic, we first fix a **formal language**. Let $\text{Prop} = \{p_1, \ldots, p_n\}$ be a finite set of **propositional variables**. Throughout this module we assume $\text{Prop}$ is finite (which simplifies some proofs), but $n$ can be chosen as large as needed.

**Definition 1.1 (Well-formed Formulas).** Given $\text{Prop} = \{p_1, \ldots, p_n\}$, we define the set $\textit{Fml}$ of all **(well-formed) formulas** inductively:

- If $p \in \text{Prop}$, then $p \in \textit{Fml}$;
- If $F, G \in \textit{Fml}$, then so are: $\neg F$, $(F \wedge G)$, $(F \vee G)$, $(F \to G)$.

> **Note on predicate logic:** Propositional logic and predicate logic are closely related. A propositional variable $p$ can be viewed as a predicate symbol of arity 0, whose truth value is fixed by a model irrespective of any arguments — just as in propositional logic where each variable is true or false in a model, which is just a row of a truth table.

---

### Semantics vs. Syntax

Logics can be defined in two broad ways:

| Approach | Description |
|----------|-------------|
| **Semantics** | Defines a **model** assigning truth values to atomic components; defines what it means for a formula to be "true" in that world. In propositional logic, models are captured by truth tables. In predicate logic, a richer model tells us the individual objects, and how to interpret each predicate, function, and constant symbol. |
| **Syntax** | Defines mechanical rules (a **proof system** / **deductive system**) to derive conclusions from premises based purely on formula structure. Example: Natural Deduction. |

In both approaches we ultimately obtain a **consequence relation** — so we identify a logic with a consequence relation.

---

### Models in Propositional Logic

**Definition 1.2 (Propositional Model / Valuation).** A **(propositional) model** or **valuation** is a function $V : \text{Prop} \to \{0, 1\}$ mapping each variable to a Boolean value (0 = false, 1 = true).

We extend $V$ to all of $\textit{Fml}$ inductively on the structure of the formula:

$$V(\neg \varphi) = 1 - V(\varphi)$$

$$V(\varphi \wedge \psi) = \min(V(\varphi), V(\psi))$$

$$V(\varphi \vee \psi) = \max(V(\varphi), V(\psi))$$

$$V(\varphi \to \psi) = \max(1 - V(\varphi), V(\psi))$$

We say $V$ is a **model of** $\varphi$ if $V(\varphi) = 1$, and write:

$$\text{mod}(\varphi) = \{V : \text{Prop} \to \{0,1\} : V(\varphi) = 1\}$$

It is sometimes convenient to associate each model $V : \text{Prop} \to \{0,1\}$ with a formula $F_V \in \textit{Fml}$ such that $V'(F_V) = 1$ if and only if $V' = V$:

$$F_V = \bigwedge_{p \in \text{Prop}} p^{V(p)}$$

where $p^1 = p$ and $p^0 = \neg p$, for $p \in \text{Prop}$.

**Example:** If $V(p) = 0$, $V(q) = 1$, $V(r) = 0$, then $F_V = \neg p \wedge q \wedge \neg r$.

---

### Natural Deduction (Syntax)

**Definition 1.3 (Natural Deduction Rule).** A **(natural deduction) rule** has the form:

$$\frac{F_1 \quad F_2 \quad \ldots \quad F_n}{G}$$

where $F_1, \ldots, F_n \in \textit{Fml}$ are the **premises** and $G \in \textit{Fml}$ is the **conclusion**.

A **(natural deduction) proof system** $\mathcal{ND}$ is a finite set of such rules. A **proof** in $\mathcal{ND}$ is a sequence of formulas where each formula is either an assumption/input, or the conclusion of some rule instance whose premises all appear earlier in the sequence.

---

### What is a Logic?

**Definition 1.4 (Logic / Consequence Relation).** Given a set of formulas $\textit{Fml}$ generated by some grammar, we identify a **logic** with a binary **consequence relation**:

$${\models} \;\subseteq\; \mathcal{P}(\textit{Fml}) \times \textit{Fml}$$

between sets of formulas from $\mathcal{P}(\textit{Fml})$ (the powerset of all subsets of $\textit{Fml}$) and individual formulas from $\textit{Fml}$, often satisfying some desirable properties.

We write $\Gamma \models \varphi$ (infix notation) to indicate that the pair $(\Gamma, \varphi)$ belongs to the consequence relation. This is analogous to how we write $x < y$ instead of $(x,y) \in {<}$.

**Definition 1.5 (Logical Consequence and Provability).**

- $\varphi$ is a **logical consequence** of $\Gamma \subseteq \textit{Fml}$ (written $\Gamma \models \varphi$) if every model of all formulas in $\Gamma$ is also a model of $\varphi$:

$$\Gamma \models \varphi \iff \text{mod}(\Gamma) \subseteq \text{mod}(\varphi)$$

- $\varphi$ is **provable/derivable** from $\Gamma$ (written $\Gamma \vdash \varphi$) if there is a finite proof of $\varphi$ using some finite subset of $\Gamma$ as initial assumptions:

$$\Gamma \vdash \varphi \iff \text{There is a proof of } \varphi \text{ from } \Gamma$$

**Key distinction in flavour:**
- $\Gamma \models \varphi$ is a **universal statement**: *all* models satisfying $\Gamma$ must satisfy $\varphi$.
- $\Gamma \vdash \varphi$ is an **existential statement**: there *exists* some finite proof of $\varphi$ from *some* finite subset of $\Gamma$.

We use $\models$ for semantically defined consequence and $\vdash$ for syntactically defined provability. It is quite remarkable that for propositional and predicate logic, both definitions agree on the same consequence relation.

**Theorem 1.6 (Completeness Theorem for Propositional Logic).** Let $\Gamma \subseteq \textit{Fml}$ and $F \in \textit{Fml}$, then:

$$\Gamma \models F \iff \Gamma \vdash F$$

That is, ${\models} \;=\; {\vdash}$. Although logical consequence (universal/semantic) and provability (existential/syntactic) differ considerably in character, they ultimately define the **same** consequence relation.

---

### Why Non-classical Logics?

Propositional and predicate logics are well suited to the abstract, rigid world of mathematics. However, outside this "Platonic realm" of abstract ideals, classical logic can break down or become unnecessarily cumbersome — and is clearly not how we typically reason about the world. To design automated systems to reason about the real world and interact with human reasoners, we need them to reason with **non-classical logic**.

---

## 1.2 Motivation for Non-monotonic Logics

### The Tweety Example

Consider the following argument:

- **Premise 1:** Tweety is a bird
- **Premise 2:** Birds can fly
- **Conclusion:** Tweety can fly

This can be formalised in propositional logic with $t$ = "I am Tweety", $b$ = "I am a bird", $f$ = "I can fly":

$$t \to b,\; b \to f \;\models\; t \to f$$

This is a valid logical consequence. Now suppose we also learn:

- **Premise 3:** Tweety is a penguin ($t \to p$)
- **Premise 4:** Penguins can't fly ($p \to \neg f$)

We may no longer be committed to "Tweety can fly". Yet classical logic is **monotonic** — adding premises cannot invalidate conclusions already drawn. So the conclusion $t \to f$ persists, and we also derive $t \to \neg f$, yielding the contradiction that Tweety can and cannot fly:

$$t \to b,\; b \to f,\; t \to p,\; p \to \neg f,\; t \;\models\; a$$

for any statement $a$ whatsoever — for example, "Tyrannosaurs make excellent accountants". This is the **principle of explosion**.

---

### Monotonicity: Definition

**Definition 1.7 (Monotonic Logic).** A logic is **monotonic** if, for all $\Gamma, \Delta \subseteq \textit{Fml}$ and $F \in \textit{Fml}$:

$$\Gamma \models F \quad \text{implies} \quad \Gamma \cup \Delta \models F$$

In other words, adding additional premises to an argument **cannot** invalidate any conclusions already drawn. Both classical propositional and predicate logic are monotonic.

---

### Defeasible Inference and Non-monotonic Logic

The problematic premise is "Birds can fly". Rather than removing this useful rule, we should have said "Birds can **usually** fly". What we need are **defeasible inferences** — rules that usually hold but may fail in exceptional cases.

**Non-monotonic logics** allow us to deal with potentially conflicting information and reason with defeasible inferences. To emphasise non-monotonicity, we use the symbol $|{\sim}$ to denote the consequence relation of a non-monotonic logic, with the interpretation:

$$\Gamma \,|{\sim}\, F \iff F \text{ \textit{usually} follows from } \Gamma$$

for all $\Gamma \subseteq \textit{Fml}$ and $F \in \textit{Fml}$. Throughout, we focus on the case where $\Gamma$ contains a single formula $F$, writing $F \,|{\sim}\, G$.

**Approach for non-monotonic logics:** In contrast to classical logic (define semantics first, seek a sound and complete proof system), for non-monotonic logics — where we don't have an immediate notion of what a non-monotonic model should look like — it is more reasonable to start by defining a **proof system** that captures how we want the consequence relation to behave, and then seek a corresponding semantics.

---

## 1.3 Interlude: Sequent Calculi

### Alternative Proof Systems

Beyond Natural Deduction, other proof systems exist in the land of syntax:

- **Hilbert systems** (favoured by David Hilbert): proofs are sequences of formulas, each either an axiom or the result of one of a very small number of logical rules. For classical propositional logic, the only rule is **modus ponens** $F, F \to G / G$; for predicate logic, modus ponens plus **universal generalisation** $F(x) / \forall x\, F(x)$. The axioms required for soundness and completeness are considerably more opaque.
- **Sequent Calculi** (also called **Gentzen systems**, after Gerhard Gentzen who first proposed them in 1935): rules operate on objects called **sequents**, not individual formulas.

---

### Sequents

**Definition 1.8 (Sequent).** A **sequent** is a pair of **finite** sets (or lists) of formulas from $\textit{Fml}$, written:

$$A_1, \ldots, A_n \;\Rightarrow\; B_1, \ldots, B_m$$

where $A_1, \ldots, A_n, B_1, \ldots, B_m \in \textit{Fml}$.

**Intuitive reading:** "If **everything** on the left is true, then **something** on the right is true."

This is purely a design choice which then motivates the kinds of rules we introduce. A Sequent Calculus can be thought of as a kind of *meta-proof* that operates directly on the consequence relation between finite sets of formulas, in contrast to Natural Deduction or Hilbert systems that operate at the level of individual formulas.

---

### Sequent Rules and the Sequent Calculus $\mathcal{S}$

**Definition 1.9 (Sequent Rule and Sequent Calculus).** A **sequent rule** takes as input zero or more sequents as **premises** and returns a single sequent as its **conclusion**:

$$\frac{\Gamma_1 \Rightarrow \Delta_1 \quad \ldots \quad \Gamma_n \Rightarrow \Delta_n}{\Gamma \Rightarrow \Delta}$$

A rule may have one or more **side conditions** that stipulate additional criteria under which the rule may be applied. A sequent rule with **zero premises** is an **axiom schema**. A **sequent calculus** is any finite set $\mathcal{S}$ of sequent rules.

Each rule acts as a **template** that can be **instantiated** with arbitrary formulas fitting the same syntactic pattern — such instantiations are called **instances** of the rule.

---

### The Sequent Calculus $\mathcal{S}$ for Classical Propositional Logic

The following set of rules $\mathcal{S}$ is adapted from Gentzen [2]. There is a single **Reflexivity axiom (REF)**, and a *left* and *right* rule for each of the Boolean connectives $\wedge, \vee, \neg, \to$, together with structural rules. Where $F, G \in \textit{Fml}$ are arbitrary formulas, and $\Gamma, \Delta$ stand for finite sets (or sequences) of formulas that are not **active** in the application of the rule.

#### Reflexivity (Axiom Schema)

$$\frac{\varnothing}{F \Rightarrow F} \quad \textbf{(REF)}$$

*For any formula $F$, we can always infer a sequent in which $F$ appears on both sides — i.e., $F$ always implies itself.*

#### Conjunction Rules

$$\frac{\Gamma, F, G \Rightarrow \Delta}{\Gamma, (F \wedge G) \Rightarrow \Delta} \;\textbf{(L}\wedge\textbf{)} \qquad \frac{\Gamma \Rightarrow \Delta, F \quad \Gamma \Rightarrow \Delta, G}{\Gamma \Rightarrow \Delta, (F \wedge G)} \;\textbf{(R}\wedge\textbf{)}$$

- **(L$\wedge$):** If $F$ and $G$ appear on the left, they can be joined into $F \wedge G$.
- **(R$\wedge$):** To prove $F \wedge G$ on the right from a context, it suffices to prove $F$ on the right and $G$ on the right separately.

#### Disjunction Rules

$$\frac{\Gamma, F \Rightarrow \Delta \quad \Gamma, G \Rightarrow \Delta}{\Gamma, (F \vee G) \Rightarrow \Delta} \;\textbf{(L}\vee\textbf{)} \qquad \frac{\Gamma \Rightarrow \Delta, F, G}{\Gamma \Rightarrow \Delta, (F \vee G)} \;\textbf{(R}\vee\textbf{)}$$

- **(L$\vee$):** If the same goal follows from $F$ alone and from $G$ alone, it follows from $F \vee G$.
- **(R$\vee$):** If $F$ and $G$ both appear on the right, they can be joined into $F \vee G$.

#### Negation Rules

$$\frac{\Gamma \Rightarrow \Delta, F}{\Gamma, \neg F \Rightarrow \Delta} \;\textbf{(L}\neg\textbf{)} \qquad \frac{\Gamma, F \Rightarrow \Delta}{\Gamma \Rightarrow \Delta, \neg F} \;\textbf{(R}\neg\textbf{)}$$

- **(L$\neg$):** Move any formula from the right to the left by adding a negation.
- **(R$\neg$):** Move any formula from the left to the right by adding a negation.

#### Implication Rules

$$\frac{\Gamma \Rightarrow \Delta, F \quad \Gamma, G \Rightarrow \Delta}{\Gamma, (F \to G) \Rightarrow \Delta} \;\textbf{(L}\to\textbf{)} \qquad \frac{\Gamma, F \Rightarrow \Delta, G}{\Gamma \Rightarrow \Delta, (F \to G)} \;\textbf{(R}\to\textbf{)}$$

- **(L$\to$):** Behaves similarly to the left disjunction rule, since $F \to G \equiv \neg F \vee G$.
- **(R$\to$):** Behaves similarly to the right disjunction rule.

#### Structural Rules

$$\frac{\Gamma \Rightarrow \Delta}{\Gamma, F \Rightarrow \Delta} \;\textbf{(MON)} \qquad \frac{\Gamma_1 \Rightarrow \Delta_1, \Theta \quad \Gamma_2, \Theta \Rightarrow \Delta_2}{\Gamma_1, \Gamma_2 \Rightarrow \Delta_1, \Delta_2} \;\textbf{(CUT)}$$

- **(MON) — Monotonicity:** We can add any formula or list of formulas $\Theta$ to the left-hand side of any sequent already proved. (This captures the monotonicity of classical logic directly.)
- **(CUT):** Merge two sequents by "cutting out" any formula or formulas $\Theta$ that appear on the right of one sequent and the left of the other. $\Theta$ here can be any finite list of formulas.

---

### Proofs in $\mathcal{S}$

**Definition 1.10 (Proof / Proof Tree).** A **proof** (or **proof tree**) in a sequent calculus $\mathcal{S}$ is a (directed) tree $T = (V, E)$ in which each vertex $v \in V$ is labelled by an instance of a sequent rule from $\mathcal{S}$, such that if a vertex $v \in V$ is labelled by an instance of a rule with $n$ premises, then $v$ has $n$ children $u_1, \ldots, u_n$ with $(v, u_i) \in E$, for $i = 1, \ldots, n$, such that $u_i$ is labelled by an instance of a rule whose conclusion is the $i$-th premise of $v$.

We say that $F \in \textit{Fml}$ is **derivable** or **provable** in $\mathcal{S}$ from $\Gamma \subseteq \textit{Fml}$ if there is a proof in $\mathcal{S}$ of the sequent $\Gamma' \Rightarrow F$ for some **finite** subset $\Gamma' \subseteq \Gamma$.

We write $\Gamma \vdash^{\mathcal{S}} F$, or simply $\Gamma \vdash F$ when $\mathcal{S}$ is clear from context.

**Proof strategy:** It is common to work **backwards** from the root, starting with the sequent we intend to prove, applying rules in reverse until all leaves are instances of the REF axiom.

---

### Completeness of $\mathcal{S}$

**Theorem 1.11 (Completeness Theorem for $\mathcal{S}$).** Let $\Gamma \subseteq \textit{Fml}$ and $F \in \textit{Fml}$, then:

$$\Gamma \models F \iff \Gamma \vdash^{\mathcal{S}} F$$

where $\models$ is defined by the usual truth-table semantics. As with natural deduction, $\mathcal{S}$ is both **sound** (every provable consequence is a logical consequence) and **complete** (every logical consequence is provable).

---

### Worked Example 1: Proving $\neg p \wedge q \vdash \neg p \vee \neg q$

**Goal sequent (root):** $\neg p \wedge q \Rightarrow \neg p \vee \neg q$

Work backwards:

1. Right side has a disjunction $\neg p \vee \neg q$. Apply **(R$\vee$)**: it suffices to prove $\neg p \wedge q \Rightarrow \neg p, \neg q$ (both $\neg p$ and $\neg q$ on right simultaneously).

2. Apply **(R$\neg$)** to move $q$ from right to left (adding negation, so $\neg\neg q$ becomes $q$ on the left): gives $q, \neg p \wedge q \Rightarrow \neg p$.

3. Apply **(R$\neg$)** to move $p$ from right to left: gives $p, q, \neg p \wedge q \Rightarrow$.

4. Apply **(L$\neg$)** to move $\neg p \wedge q$ — split using **(L$\wedge$)** to get $p, q, p, q \Rightarrow$... But more directly:

After moving $p$ and $q$ to the left, use **(R$\wedge$)** to split the goal into $p, q \Rightarrow p$ and $p, q \Rightarrow q$, each of which is an instance of **(REF)** ($p$ appears on both sides; $q$ appears on both sides).

The proof tree (working top-down from leaves):

$$\frac{\dfrac{\varnothing}{p,q \Rightarrow p}\;(\text{REF}) \qquad \dfrac{\varnothing}{p,q \Rightarrow q}\;(\text{REF})}{p,q \Rightarrow p \wedge q}\;(\text{R}\wedge)$$

then applying the negation and disjunction rules in reverse up to the root $\neg p \wedge q \Rightarrow \neg p \vee \neg q$.

---

### Worked Example 2: Proving $\neg p \vee q \vdash \neg p \wedge \neg q$

**Goal sequent (root):** $\neg p \vee q \Rightarrow \neg p \wedge \neg q$

Work backwards:

1. **(R$\wedge$)** splits: need to prove $\neg p \vee q \Rightarrow \neg p$ and $\neg p \vee q \Rightarrow \neg q$.

2. **First branch:** **(R$\neg$)** moves $p$ to left: $p, \neg p \vee q \Rightarrow \varnothing$.
   **Second branch:** **(R$\neg$)** moves $q$ to left: $q, \neg p \vee q \Rightarrow \varnothing$.

3. In both branches, apply **(L$\neg$)** to move $\neg p \vee q$ to the right: first branch becomes $p \Rightarrow p \vee q$; second branch becomes $q \Rightarrow p \vee q$.

4. Apply **(R$\vee$)** to split the disjunction on the right in both branches: each becomes a **(REF)** instance. All leaves are REF axioms — proof complete.

---

### Derived Rule: Double Negation Elimination (Tutorial Q1.1)

The following rules of **double negation elimination** are derivable in $\mathcal{S}$:

$$\frac{\Gamma \Rightarrow \Delta, \neg\neg F}{\Gamma \Rightarrow \Delta, F} \qquad \text{and} \qquad \frac{\Gamma, \neg\neg F \Rightarrow \Delta}{\Gamma, F \Rightarrow \Delta}$$

**Proof of Rule 1** (given $\Gamma \Rightarrow \Delta, \neg\neg F$, derive $\Gamma \Rightarrow \Delta, F$):

$$\dfrac{\dfrac{\dfrac{\varnothing}{F \Rightarrow F}\;(\text{REF})}{\varnothing \Rightarrow \neg F, F}\;(\text{R}\neg)}{\neg\neg F \Rightarrow F}\;(\text{L}\neg) \qquad \Gamma \Rightarrow \Delta, \neg\neg F}{\Gamma \Rightarrow \Delta, F}\;(\text{CUT})$$

**Proof of Rule 2** (given $\Gamma, \neg\neg F \Rightarrow \Delta$, derive $\Gamma, F \Rightarrow \Delta$):

$$\dfrac{\dfrac{\dfrac{\varnothing}{F \Rightarrow F}\;(\text{REF})}{\neg F, F \Rightarrow \varnothing}\;(\text{L}\neg)}{F \Rightarrow \neg\neg F}\;(\text{R}\neg) \qquad \Gamma, \neg\neg F \Rightarrow \Delta}{\Gamma, F \Rightarrow \Delta}\;(\text{CUT})$$

## 1.4 The Gabbay–Makinson Rules

### Overview and Motivation

In contrast to classical (monotonic) propositional logic — where we first defined a semantics (truth tables) and then sought a proof system that was sound and complete with respect to the semantics — for non-monotonic logics, we start by defining the **syntactic rules** that we expect any reasonable interpretation of "usually" should follow.

One such set of rules are those proposed by **Dov Gabbay** and **David Makinson** [1, 3]. We call these the **Gabbay–Makinson rules**, forming the proof system $\mathcal{GM}$.

To distinguish from the $\Rightarrow$ sequents of $\mathcal{S}$, we use $|{\sim}$ for the sequents of $\mathcal{GM}$, read colloquially as **"F snakes G"**. We focus on the case of a single formula on the left: $F \,|{\sim}\, G$.

We say $G$ is **provable/derivable** from $F$ in $\mathcal{GM}$ if there is a proof of the sequent $F \,|{\sim}\, G$. We write $F \,|{\sim}^{\mathcal{GM}}\, G$, or simply $F \,|{\sim}\, G$ when clear from context.

---

### The Rules of $\mathcal{GM}$

The Gabbay–Makinson proof system $\mathcal{GM}$ comprises the following rules, where $F, G, H \in \textit{Fml}$ are arbitrary formulas:

#### Rule 1: Reflexivity (REF) — Axiom Schema

$$\frac{\varnothing}{F \,|{\sim}\, F} \quad \textbf{(REF)}$$

**Meaning:** $F$ usually follows from $F$, for any formula $F$. We are always allowed to infer that $F$ usually follows from $F$. This mirrors the reflexivity axiom of $\mathcal{S}$.

---

#### Rule 2: Right Conjunction (R$\wedge$)

$$\frac{F \,|{\sim}\, G \quad F \,|{\sim}\, H}{F \,|{\sim}\, (G \wedge H)} \quad \textbf{(R}\wedge\textbf{)}$$

**Meaning:** If $G$ and $H$ both usually follow from $F$, then their conjunction $(G \wedge H)$ usually follows from $F$. Borrowed from the right conjunction rule of propositional logic.

---

#### Rule 3: Left Disjunction (L$\vee$)

$$\frac{F \,|{\sim}\, H \quad G \,|{\sim}\, H}{(F \vee G) \,|{\sim}\, H} \quad \textbf{(L}\vee\textbf{)}$$

**Meaning:** If $H$ usually follows from $F$ and $H$ usually follows from $G$, then it should not matter which of $F$ or $G$ is true for $H$ to usually follow — that is, $H$ usually follows from $F \vee G$. Borrowed from the left disjunction rule of propositional logic.

---

#### Rule 4: Left Logical Equivalence (LLE)

$$\frac{F \,|{\sim}\, H \quad F \equiv G}{G \,|{\sim}\, H} \quad \textbf{(LLE)}$$

**Meaning:** If $H$ usually follows from $F$, and we also know that $F$ is **classically logically equivalent** to $G$ (i.e., $F \equiv G$ according to classical logic), then we can infer that $H$ usually follows from $G$.

> **Strictness note:** To avoid mixing classical logic and non-monotonic logic, the condition $F \equiv G$ should be considered as a **side condition** under which the rule is applicable; it is notationally convenient to write it as a premise.

---

#### Rule 5: Right Weakening (RWE)

$$\frac{F \,|{\sim}\, G \quad G \models H}{F \,|{\sim}\, H} \quad \textbf{(RWE)}$$

**Meaning:** If $G$ usually follows from $F$, and $H$ is a classical logical consequence of $G$ (i.e., $G \models H$, so $H$ is a logically weaker statement), then $H$ usually follows from $F$.

> **Note:** Unlike (LLE), this does not require $G \equiv H$ — only that $H$ can be classically inferred from $G$.

---

#### Rule 6: Cautious Monotonicity (CMON)

$$\frac{F \,|{\sim}\, G \quad F \,|{\sim}\, H}{(F \wedge G) \,|{\sim}\, H} \quad \textbf{(CMON)}$$

**Meaning:** If $H$ usually follows from $F$, then $H$ also usually follows from $F \wedge G$, **provided** that $G$ also usually follows from $F$. The idea is that $F \wedge G$ is not too unlikely a scenario when $G$ is usually present whenever $F$ is true. This is a **weakened form of monotonicity** — we can add conjuncts, but only those that usually hold.

> **Note:** Gabbay refers to this as "Restricted Monotonicity".

---

#### Rule 7: Rational Monotonicity (RMON)

$$\frac{F \,|{\sim}\, H \quad F \,\not|{\sim}\, \neg G}{(F \wedge G) \,|{\sim}\, H} \quad \textbf{(RMON)}$$

**Meaning:** If $H$ usually follows from $F$, then $H$ also usually follows from $F \wedge G$, **provided** that $\neg G$ does **not** usually follow from $F$.

**Key difference from CMON:** RMON does not require that $G$ usually follows from $F$ — only that $\neg G$ does *not* usually follow. It could be that **neither** $G$ nor $\neg G$ usually follows from $F$.

**Illustrative example:** Let $F$ = "I flip a fair coin" and $G$ = "the coin lands on heads". Then clearly neither $G$ nor $\neg G$ usually follows from $F$. So $F \,\not|{\sim}\, \neg G$, meaning we can apply RMON.

> **Special feature of RMON:** Unlike the other rules, one of its premises requires proving that something **does not** usually follow — a negative premise.

---

### Summary Table of $\mathcal{GM}$ Rules

| Rule | Name | Premises | Conclusion |
|------|------|----------|------------|
| **(REF)** | Reflexivity | (none) | $F \,|{\sim}\, F$ |
| **(R$\wedge$)** | Right Conjunction | $F \,|{\sim}\, G$; $F \,|{\sim}\, H$ | $F \,|{\sim}\, (G \wedge H)$ |
| **(L$\vee$)** | Left Disjunction | $F \,|{\sim}\, H$; $G \,|{\sim}\, H$ | $(F \vee G) \,|{\sim}\, H$ |
| **(LLE)** | Left Logical Equivalence | $F \,|{\sim}\, H$; $F \equiv G$ | $G \,|{\sim}\, H$ |
| **(RWE)** | Right Weakening | $F \,|{\sim}\, G$; $G \models H$ | $F \,|{\sim}\, H$ |
| **(CMON)** | Cautious Monotonicity | $F \,|{\sim}\, G$; $F \,|{\sim}\, H$ | $(F \wedge G) \,|{\sim}\, H$ |
| **(RMON)** | Rational Monotonicity | $F \,|{\sim}\, H$; $F \,\not|{\sim}\, \neg G$ | $(F \wedge G) \,|{\sim}\, H$ |

## 1.5 Some Derived Rules of $\mathcal{GM}$

As with Natural Deduction, it is often convenient to work not only with the basic rules of $\mathcal{GM}$, but also with **derived rules** that can be derived from the basic rules. The following derived rules can each be derived from the basic rules of $\mathcal{GM}$, and do not add expressive power but simplify proofs.

---

### Derived Rule 1: Superclassicality (SC)

$$\frac{F \models G}{F \,|{\sim}\, G} \quad \textbf{(SC)}$$

**Meaning:** If $G$ always follows from $F$ (classical logical consequence), then $G$ usually follows from $F$. Classical consequences are also defeasible consequences.

**Derivation from basic rules:**
1. By **(REF)**: $F \,|{\sim}\, F$.
2. Since $F \models G$, by **(RWE)**: $F \,|{\sim}\, G$. $\square$

---

### Derived Rule 2: Conditionalisation (CON)

$$\frac{(F \wedge G) \,|{\sim}\, H}{F \,|{\sim}\, (G \to H)} \quad \textbf{(CON)}$$

**Meaning:** If $H$ usually follows from $F$ and $G$ together, then the implication $(G \to H)$ usually follows from $F$ alone. This moves a conjunct from the left-hand side into an implication on the right — analogous to the **(R$\to$)** rule of $\mathcal{S}$.

**Derivation from basic rules:** Uses **(SC)**, **(RWE)**, **(L$\vee$)**, and **(LLE)**.

---

### Derived Rule 3: Cautious Cut (CC)

$$\frac{F \,|{\sim}\, G \quad (F \wedge G) \,|{\sim}\, H}{F \,|{\sim}\, H} \quad \textbf{(CC)}$$

**Meaning:** If $H$ usually follows from $F$ and $G$ together, and $G$ is usually true whenever $F$ is true, then $G$ is somewhat redundant and we can conclude $H$ usually follows from $F$ alone.

**Derivation from basic rules:** Uses **(CON)**, **(R$\wedge$)**, and **(RWE)**.

---

### Derived Rule 4: Negation Rationality (NRAT)

$$\frac{F \,|{\sim}\, H \quad (F \wedge \neg G) \,\not|{\sim}\, H}{(F \wedge G) \,|{\sim}\, H} \quad \textbf{(NRAT)}$$

**Meaning:** If $H$ usually follows from $F$, but does not usually follow from $F$ together with $\neg G$, then $H$ should follow from $F$ together with $G$.

**Contrapositive form (NRAT)':**

$$\frac{(F \wedge G) \,\not|{\sim}\, H \quad (F \wedge \neg G) \,\not|{\sim}\, H}{F \,\not|{\sim}\, H} \quad \textbf{(NRAT)}'$$

Meaning: if $H$ doesn't follow from $F \wedge G$ or from $F \wedge \neg G$, then there must be something inherent about $F$ alone that means $H$ doesn't usually follow from $F$.

**Derivation from basic rules:** Uses **(CMON)**, **(RMON)**, and **(LLE)**.

---

### Summary of Derived Rules

| Rule | Name | Premises | Conclusion |
|------|------|----------|------------|
| **(SC)** | Superclassicality | $F \models G$ | $F \,|{\sim}\, G$ |
| **(CON)** | Conditionalisation | $(F \wedge G) \,|{\sim}\, H$ | $F \,|{\sim}\, (G \to H)$ |
| **(CC)** | Cautious Cut | $F \,|{\sim}\, G$; $(F \wedge G) \,|{\sim}\, H$ | $F \,|{\sim}\, H$ |
| **(NRAT)** | Negation Rationality | $F \,|{\sim}\, H$; $(F \wedge \neg G) \,\not|{\sim}\, H$ | $(F \wedge G) \,|{\sim}\, H$ |

## 1.6 Worked Examples with $\mathcal{GM}$

### The Penguin/Bird Knowledge Base

Let $b$ = "is a bird", $p$ = "is a penguin", $f$ = "can fly". The three defeasible premises are:

1. $b \,|{\sim}\, f$ — "Birds usually fly"
2. $p \,|{\sim}\, \neg f$ — "Penguins usually don't fly"
3. $p \wedge \neg b \,|{\sim}\, \bot$ — "Penguins that are not birds are usually impossible" (i.e., penguins are birds)

---

### Example 1: $b \wedge p \,|{\sim}\, \neg f$ — "Birds that are penguins usually don't fly"

**Goal:** $b \wedge p \,|{\sim}\, \neg f$

We work backwards from the root.

**Step 1:** By **(CMON)**, it suffices to derive:
- $p \,|{\sim}\, \neg f$ (premise 2 — taken as an axiom, leaf of proof tree), and
- $p \,|{\sim}\, b$

**Step 2:** Derive $p \,|{\sim}\, b$:
- From premise 3: $p \wedge \neg b \,|{\sim}\, \bot$
- Apply **(CON)** (conditionalisation): $p \,|{\sim}\, (\neg b \to \bot)$
- Classically, $(\neg b \to \bot) \equiv b$, so $(\neg b \to \bot) \models b$
- Apply **(RWE)**: $p \,|{\sim}\, b$ ✓

**Step 3:** Now by **(CMON)** with $p \,|{\sim}\, \neg f$ and $p \,|{\sim}\, b$:
$$(p \wedge b) \,|{\sim}\, \neg f$$

**Step 4:** Classically $p \wedge b \equiv b \wedge p$. Apply **(LLE)**:
$$b \wedge p \,|{\sim}\, \neg f \quad \checkmark$$

---

### Example 2: $b \,|{\sim}\, \neg p$ — "Birds are usually not penguins"

**Goal:** $b \,|{\sim}\, \neg p$

**Step 1:** From Example 1, we have $b \wedge p \,|{\sim}\, \neg f$.
Apply **(CON)**: $b \,|{\sim}\, (p \to \neg f)$.

**Step 2:** From premise 1: $b \,|{\sim}\, f$.
Apply **(R$\wedge$)**: $b \,|{\sim}\, (p \to \neg f) \wedge f$.

**Step 3:** Classically, $(p \to \neg f) \wedge f \models \neg p$ (if $p \to \neg f$ and $f$, then $p$ must be false).
Apply **(RWE)**: $b \,|{\sim}\, \neg p$. $\checkmark$

Since all leaves of the proof tree are either premises or results already proved from premises, the conclusion is a rational consequence of the premises.

### Large Group Tutorial Q1.3: Disjunctive Rationality (DRAT)

The following derived rule of **Disjunctive Rationality** is derivable in $\mathcal{GM}$:

$$\frac{(F \vee G) \,|{\sim}\, H \quad F \,\not|{\sim}\, H}{G \,|{\sim}\, H} \quad \textbf{(DRAT)}$$

**Symmetrically stated:** $F \,\not|{\sim}\, H$ and $G \,\not|{\sim}\, H$ implies $(F \vee G) \,\not|{\sim}\, H$.

**Full proof** — assume $(F \vee G) \,|{\sim}\, H$ and $F \,\not|{\sim}\, H$:

**(i) Show $(F \vee G) \,|{\sim}\, \neg F$:**

Use the contrapositive of **(RMON)**: $A \,|{\sim}\, C$ and $(A \wedge B) \,\not|{\sim}\, C$ implies $A \,|{\sim}\, \neg B$.

Take $A = (F \vee G)$, $B = F$, $C = H$:
- $(F \vee G) \,|{\sim}\, H$ (given).
- $(F \vee G) \wedge F \equiv F$ classically; by **(LLE)** and $F \,\not|{\sim}\, H$: $(F \vee G) \wedge F \,\not|{\sim}\, H$.
- By **(RMON)** (contrapositive form): $(F \vee G) \,|{\sim}\, \neg F$. ✓

The formal tree:
$$\frac{(F \vee G) \,|{\sim}\, H \quad \dfrac{F \,\not|{\sim}\, H \quad (F \vee G) \wedge F \equiv F}{(F \vee G) \wedge F \,\not|{\sim}\, H}\;(\text{LLE})}{(F \vee G) \,|{\sim}\, \neg F}\;(\text{RMON})$$

**(ii) Show $(F \vee G) \,|{\sim}\, G$:**

- From (i): $(F \vee G) \,|{\sim}\, \neg F$.
- By **(REF)**: $(F \vee G) \,|{\sim}\, (F \vee G)$.
- By **(RMON)**: $(F \vee G) \,|{\sim}\, (F \vee G) \wedge \neg F$.
- Since $(F \vee G) \wedge \neg F \models G$ classically, by **(RWE)**: $(F \vee G) \,|{\sim}\, G$. ✓

$$\frac{\dfrac{\varnothing}{(F \vee G) \,|{\sim}\, (F \vee G)}\;(\text{REF}) \quad \text{From (i): } (F \vee G) \,\not|{\sim}\, \neg F}{(F \vee G) \,|{\sim}\, (F \vee G) \wedge \neg F}\;(\text{RMON}) \quad (F \vee G) \wedge \neg F \models G}{(F \vee G) \,|{\sim}\, G}\;(\text{RWE})$$

**(iii) Show $G \,|{\sim}\, H$:**

- From (ii): $(F \vee G) \,|{\sim}\, G$.
- Given: $(F \vee G) \,|{\sim}\, H$.
- By **(CMON)**: $(F \vee G) \wedge G \,|{\sim}\, H$.
- Since $(F \vee G) \wedge G \equiv G$ classically, by **(LLE)**: $G \,|{\sim}\, H$. ✓

$$\frac{(F \vee G) \,|{\sim}\, H \quad \text{From (ii): } F \vee G \,|{\sim}\, G}{(F \vee G) \wedge G \,|{\sim}\, H}\;(\text{CMON}) \quad (F \vee G) \wedge G \equiv G}{G \,|{\sim}\, H}\;(\text{LLE})$$

---

### Large Group Tutorial Q1.4: "If $F \vee G$ is usually impossible, then $F$ is usually impossible"

$$\frac{(F \vee G) \,|{\sim}\, \bot}{F \,|{\sim}\, \bot}$$

**Proof** — assume $(F \vee G) \,|{\sim}\, \bot$:

**(i)** Since $\bot \models (F \vee G) \to F$ (anything follows from $\bot$), by **(RWE)**:
$$(F \vee G) \,|{\sim}\, (F \vee G) \to F$$

$$\frac{(F \vee G) \,|{\sim}\, \bot \quad \bot \models (F \vee G) \to F}{(F \vee G) \,|{\sim}\, (F \vee G) \to F}\;(\text{RWE})$$

**(ii)** By **(REF)**: $(F \vee G) \,|{\sim}\, (F \vee G)$. By **(R$\wedge$)** with (i):
$(F \vee G) \,|{\sim}\, ((F \vee G) \to F) \wedge (F \vee G)$.
Since $((F \vee G) \to F) \wedge (F \vee G) \models F$, by **(RWE)**: $(F \vee G) \,|{\sim}\, F$.

$$\frac{\text{From (i)}\quad (F\!\vee\! G)\,|{\sim}\,(F\!\vee\! G)\to F \qquad \dfrac{\varnothing}{(F\!\vee\! G)\,|{\sim}\,(F\!\vee\! G)}\;(\text{REF})}{(F\!\vee\! G)\,|{\sim}\,((F\!\vee\! G)\to F)\wedge(F\!\vee\! G)}\;(\text{R}\wedge) \quad ((F\!\vee\! G)\to F)\wedge(F\!\vee\! G)\models F}{(F \vee G)\,|{\sim}\, F}\;(\text{RWE})$$

**(iii)** By **(CMON)** (using $(F \vee G) \,|{\sim}\, \bot$ and $(F \vee G) \,|{\sim}\, F$):
$(F \vee G) \wedge F \,|{\sim}\, \bot$.
Since $(F \vee G) \wedge F \equiv F$ classically, by **(LLE)**: $F \,|{\sim}\, \bot$. ✓

$$\frac{(F \vee G)\,|{\sim}\,\bot \quad \text{From (ii): }(F \vee G)\,|{\sim}\, F}{(F\vee G)\wedge F\,|{\sim}\,\bot}\;(\text{CMON}) \quad (F\vee G)\wedge F\equiv F}{F\,|{\sim}\,\bot}\;(\text{LLE})$$

### Large Group Tutorial Q1.5: Derived Rule

$$\frac{F \,|{\sim}\, G}{(F \vee H) \,|{\sim}\, (\neg F \vee G)}$$

**Proof** — assume $F \,|{\sim}\, G$:

$$\frac{F\,|{\sim}\, G \quad \dfrac{F \models (F \vee H)}{F\,|{\sim}\,(F \vee H)}\;(\text{SC})}{(F \vee H) \wedge F\,|{\sim}\, G}\;(\text{CMON}) \qquad (F \vee H)\,|{\sim}\,(F \to G)\;(\text{CON}) \quad F\to G\equiv \neg F\vee G}{(F \vee H)\,|{\sim}\,(\neg F \vee G)}\;(\text{RWE})$$

Step-by-step:
1. Since $F \models (F \vee H)$, by **(SC)**: $F \,|{\sim}\, (F \vee H)$.
2. By **(CMON)**: $(F \vee H) \wedge F \,|{\sim}\, G$.
3. By **(CON)**: $(F \vee H) \,|{\sim}\, (F \to G)$.
4. Since $F \to G \equiv \neg F \vee G$ classically, by **(RWE)**: $(F \vee H) \,|{\sim}\, (\neg F \vee G)$. ✓

## Key Definitions

| Term | Definition |
|------|------------|
| **Propositional variable** | An atomic symbol $p \in \text{Prop}$ whose truth value is either 0 (false) or 1 (true) in a model |
| **Well-formed formula (wff)** | A formula built inductively from propositional variables using $\neg, \wedge, \vee, \to$ |
| **Model / Valuation** | A function $V : \text{Prop} \to \{0,1\}$ assigning Boolean values to propositional variables |
| **$\text{mod}(\varphi)$** | The set of all models of $\varphi$: $\{V : \text{Prop} \to \{0,1\} : V(\varphi) = 1\}$ |
| **$F_V$** | The formula uniquely identifying model $V$: the conjunction $\bigwedge_{p\in\text{Prop}} p^{V(p)}$ |
| **Logic** | A consequence relation ${\models} \subseteq \mathcal{P}(\textit{Fml}) \times \textit{Fml}$ over a formal language |
| **Logical consequence ($\models$)** | $\Gamma \models \varphi$ iff every model of $\Gamma$ is a model of $\varphi$ — a universal/semantic statement |
| **Provability ($\vdash$)** | $\Gamma \vdash \varphi$ iff there exists a finite proof of $\varphi$ from $\Gamma$ — an existential/syntactic statement |
| **Completeness Theorem** | For propositional logic: $\Gamma \models F \iff \Gamma \vdash F$ (semantic = syntactic) |
| **Monotonic logic** | A logic where $\Gamma \models F$ implies $\Gamma \cup \Delta \models F$ for all $\Delta$ — adding premises never invalidates conclusions |
| **Principle of explosion** | From a contradiction, any statement can be derived in classical logic |
| **Non-monotonic logic** | A logic where adding premises can retract conclusions; models defeasible inference |
| **Defeasible inference** | A conclusion that *usually* holds but may fail in exceptional circumstances |
| **$|{\sim}$** | The non-monotonic consequence relation; $F \,|{\sim}\, G$ means "$G$ usually follows from $F$" |
| **Hilbert system** | Proof system where proofs are sequences of formulas (axioms or rule applications); uses very few rules |
| **Sequent** | A pair of finite sets of formulas $A_1,\ldots,A_n \Rightarrow B_1,\ldots,B_m$; reads "if everything on left is true, something on right is true" |
| **Sequent rule** | A rule taking zero or more sequents as premises and returning one sequent as conclusion |
| **Axiom schema** | A sequent rule with zero premises (e.g., REF) |
| **Sequent calculus / Gentzen system** | A finite set $\mathcal{S}$ of sequent rules; introduced by Gerhard Gentzen in 1935 |
| **Proof tree** | A directed tree where each node is labelled by a rule instance; leaves are axiom (REF) instances |
| **$\Gamma \vdash^{\mathcal{S}} F$** | $F$ is derivable from $\Gamma$ in $\mathcal{S}$: there is a proof tree in $\mathcal{S}$ of $\Gamma' \Rightarrow F$ for some finite $\Gamma' \subseteq \Gamma$ |
| **$\mathcal{GM}$** | The Gabbay–Makinson proof system for non-monotonic logic |
| **(REF)** | Reflexivity: $F \,|{\sim}\, F$ — axiom schema, no premises |
| **(R$\wedge$)** | Right Conjunction: $F\,|{\sim}\,G$ and $F\,|{\sim}\,H$ give $F\,|{\sim}\,G\wedge H$ |
| **(L$\vee$)** | Left Disjunction: $F\,|{\sim}\,H$ and $G\,|{\sim}\,H$ give $(F\vee G)\,|{\sim}\,H$ |
| **(LLE)** | Left Logical Equivalence: $F\,|{\sim}\,H$ and $F\equiv G$ give $G\,|{\sim}\,H$ |
| **(RWE)** | Right Weakening: $F\,|{\sim}\,G$ and $G\models H$ give $F\,|{\sim}\,H$ |
| **(CMON)** | Cautious Monotonicity: $F\,|{\sim}\,G$ and $F\,|{\sim}\,H$ give $(F\wedge G)\,|{\sim}\,H$ |
| **(RMON)** | Rational Monotonicity: $F\,|{\sim}\,H$ and $F\,\not|{\sim}\,\neg G$ give $(F\wedge G)\,|{\sim}\,H$ |
| **(SC)** | Superclassicality (derived): $F\models G$ gives $F\,|{\sim}\,G$ |
| **(CON)** | Conditionalisation (derived): $(F\wedge G)\,|{\sim}\,H$ gives $F\,|{\sim}\,(G\to H)$ |
| **(CC)** | Cautious Cut (derived): $F\,|{\sim}\,G$ and $(F\wedge G)\,|{\sim}\,H$ give $F\,|{\sim}\,H$ |
| **(NRAT)** | Negation Rationality (derived): $F\,|{\sim}\,H$ and $(F\wedge\neg G)\,\not|{\sim}\,H$ give $(F\wedge G)\,|{\sim}\,H$ |
| **(DRAT)** | Disjunctive Rationality (derived): $(F\vee G)\,|{\sim}\,H$ and $F\,\not|{\sim}\,H$ give $G\,|{\sim}\,H$ |

## Summary

This week introduced the foundational concepts of non-monotonic logics, addressing all three learning objectives:

**LO 1 — Define what it means for a logic to be (non-)monotonic:**
A logic is **monotonic** if $\Gamma \models F$ implies $\Gamma \cup \Delta \models F$ for all $\Delta$ — adding premises never invalidates conclusions. Both classical propositional and predicate logic are monotonic. The Tweety/penguin example demonstrates why monotonicity is problematic for real-world reasoning: it leads to the **principle of explosion** when conflicting information is encountered, and prevents rational retraction of conclusions. A logic is identified with a consequence relation over a formal language; a logic is **non-monotonic** when it supports **defeasible inferences** — conclusions that *usually* hold but can be retracted in the face of new information.

**LO 2 — Apply Sequent rules to construct proofs in classical propositional logic:**
The **sequent calculus $\mathcal{S}$** (adapted from Gentzen, 1935) provides a proof system operating on **sequents** $\Gamma \Rightarrow \Delta$ (read: "if everything on the left, then something on the right"). It contains one axiom schema **(REF)**, left/right rules for each connective $\wedge, \vee, \neg, \to$, and two structural rules **(MON)** and **(CUT)**. Proofs are **directed trees** built backwards from the goal sequent until all leaves are REF instances. The calculus is **sound and complete**: $\Gamma \models F \iff \Gamma \vdash^{\mathcal{S}} F$. Key sequent calculus proofs covered include: $\neg p \wedge q \vdash \neg p \vee \neg q$, $\neg p \vee q \vdash \neg p \wedge \neg q$, De Morgan laws, and double negation elimination.

**LO 3 — Apply the Gabbay–Makinson rules to construct proofs of rational consequences:**
The **Gabbay–Makinson system $\mathcal{GM}$** defines a non-monotonic consequence relation $|{\sim}$ via seven rules: **(REF)**, **(R$\wedge$)**, **(L$\vee$)**, **(LLE)**, **(RWE)**, **(CMON)**, and **(RMON)**. The approach is syntactic-first: define the rules that any reasonable "usually" relation should satisfy. Key derived rules include **Superclassicality (SC)** (classical consequences are defeasible), **Conditionalisation (CON)** (move conjuncts into implications), **Cautious Cut (CC)** (remove redundant usual premises), **Negation Rationality (NRAT)**, and **Disjunctive Rationality (DRAT)**. Proofs in $\mathcal{GM}$ are also proof trees built backwards. Core worked examples: birds that are penguins usually don't fly ($b \wedge p \,|{\sim}\, \neg f$); birds are usually not penguins ($b \,|{\sim}\, \neg p$); core modules are usually not logic modules.
